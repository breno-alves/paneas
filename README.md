## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# test coverage
$ yarn run test:cov
```

## Resources

- [GET] /api - api documentation
- [GET] /health - health check
- [GET] /joke - get a random joke
- [POST] /favorite - save a joke
