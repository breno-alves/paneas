## Installation

```bash
$ yarn install
```

---

## Running the app

### Docker

```bash
$ docker compose up -d
```


### Local

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev
```

---

## Test

```bash
# unit tests
$ yarn run test

# test coverage
$ yarn run test:cov
```
